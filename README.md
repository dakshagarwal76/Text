## ✨ Features

- 🔠 **Convert to Uppercase**
- 🔡 **Convert to Lowercase**
- 🔢 **Real-time Word & Character Count**
- 🧹 **Clear Text Instantly**
- 🌓 **Toggle Dark / Light Mode**
- 📱 **Responsive Design**
